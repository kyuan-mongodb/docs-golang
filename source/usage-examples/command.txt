=============
Run a Command
=============

.. default-domain:: mongodb

You can run commands directly on the MongoDB server CLI using the
``RunCommand()`` method. 

The following example passes a document containing the ``dbStats``
command to the ``RunCommand()`` method, which requests for statistics
about the ``sample_restaurants`` database:

   :emphasize-lines: 4
   :language: go
   :dedent:

Click `here <{+example+}/command.go>`__ to see a fully runnable example.

Expected Result
---------------

After running the preceding code snippet, the ``result`` variable should
contain the following: 

.. code-block:: json

    // results truncated
    [ {db sample_restaurants} 
      {collections 2} 
      {views 0} 
      {objects 25554} 
      {avgObjSize 548.4101901854896} 
      {dataSize 14014074} {storageSize 8257536} 
      ... ]

.. note::

    The ``result`` variable may vary depending on your data set.
