===============
Count Documents
===============

.. default-domain:: mongodb

You can count the number of documents in a collection using the
``EstimatedDocumentCount()`` and ``CountDocuments()`` method. 

The following example counts the number of documents in the ``movies``
collection using the ``EstimatedDocumentCount()`` method. Then, the
example counts the number of documents in the ``movies`` collection
where the ``countires`` field is "China" using the ``CountDocuments()``
method.

Expected Result
---------------

After running the preceding code snippet, you should see the following:

- There are about ``23541`` documents in the ``movies`` collection  
- There are ``755`` documents in the ``movies`` collection from "China"

.. note::

    The exact number of documents may vary depending on your data set.
